Project Overview This project focuses on extracting and visualizing stock data for companies like Tesla and GameStop. By using libraries such as yfinance for stock data extraction and web scraping techniques to retrieve revenue data, the project provides insights into the relationship between stock prices and company revenues. The extracted data is visualized using Python libraries like Plotly and Matplotlib for easy interpretation. The project contains:

A graphing function to visualize stock price and revenue. Data extraction from Yahoo Finance and MacroTrends. Graphs for Tesla and GameStop stock data.
